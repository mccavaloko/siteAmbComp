    Placa Mãe

    Placa mãe é o componente principal de um computador, pois é ela que conecta todos os outros componentes. Ela é responsável por controlar o fluxo de dados entre os componentes, além de fornecer energia para eles.

    Processador

    O processador é o cérebro do computador. Ele é responsável por executar todas as tarefas do computador, como abrir programas, navegar na internet, etc.

    Saiba mais

    Memória RAM

    A memória RAM é um componente de hardware que armazena dados e instruções que são utilizados pelo processador. Ela é responsável por armazenar informações temporariamente, ou seja, enquanto o computador estiver ligado.

    Placa de Vídeo

    A placa de vídeo é um componente de hardware que é responsável por gerar a imagem que é exibida na tela do computador
